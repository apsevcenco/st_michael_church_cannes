# Сайт прихода Архангела Михаила в Каннах

Статический прототип сайта православного прихода Архистратига Михаила в Каннах.

## Файлы

- `index.html` — главная страница сайта.
- `assets/styles.css` — стили и адаптивная сетка.
- `structure.md` — рекомендуемая структура полноценного сайта.

## Перед публикацией

Нужно заменить прототипные данные на официальные:

- актуальное расписание богослужений;
- контакты канцелярии;
- реквизиты для пожертвований;
- юридическое наименование прихода;
- разрешенные фотографии и материалы.
